# Agent Engineering Roadmap

这是一个以 Hello-Agents 为第一阶段主线的 Agent Engineering 学习仓库。

当前节奏：

- 两周内完成 Hello-Agents 主体内容。
- 每天至少一章，部分较轻章节合并。
- 每天可在本地沉淀一篇公众号草稿，副标题使用 `Agent 学习（N）`，稿件不上传 GitHub。
- 每天的代码实验放在当天 `experiments/`，阶段性作品再整理到 `projects/`。

核心入口：

- `roadmap/hello-agents-14-day-sprint.md`
- `daily/`

GitHub 仅保存学习路线、笔记、参考资料和代码实验。公众号稿件、发布产物与账号凭据只保留在本地，并由 `.gitignore` 排除。
